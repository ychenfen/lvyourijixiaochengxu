# 图片资源说明

本目录存放小程序所需的图片资源。

## TabBar 图标

需要准备以下图标（建议尺寸 81x81 像素，PNG 格式）：

- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `diary.png` - 日记图标（未选中）
- `diary-active.png` - 日记图标（选中）
- `footprint.png` - 足迹图标（未选中）
- `footprint-active.png` - 足迹图标（选中）
- `profile.png` - 我的图标（未选中）
- `profile-active.png` - 我的图标（选中）

## 其他图标

- `marker.png` - 地图标记点图标（建议尺寸 32x40 像素）
- `default-avatar.png` - 默认头像
- `default-cover.png` - 默认日记封面
- `default-place.png` - 默认商户图片

## 图标获取建议

可以从以下网站获取免费图标：
- [阿里巴巴矢量图标库](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)

下载后放入本目录即可使用。
